# Лабораторная работа №4 «Разработка web-сервиса»

## Предметная область: Продажа напитков

Веб-сервис для управления продажей напитков с использованием Flask и flask_restplus для автоматической документации API через Swagger.

## Структура данных

Модель данных напитка содержит следующие поля:
- **id** (string) - Идентификатор напитка
- **name** (string) - Название напитка
- **manufacturer** (string) - Производитель
- **type** (string) - Тип напитка (Газированный, Сок, Вода, Энергетический и т.д.)
- **volume** (float) - Объем в миллилитрах
- **price** (float) - Цена в рублях
- **stock** (integer) - Количество в наличии

## Функциональность

### CRUD операции
- **GET /beverages/** - Получить список всех напитков (с сортировкой)
- **POST /beverages/** - Добавить новый напиток
- **GET /beverages/<id>** - Получить напиток по ID
- **PUT /beverages/<id>** - Обновить напиток по ID
- **DELETE /beverages/<id>** - Удалить напиток по ID

### Сортировка
Все эндпоинты списка поддерживают сортировку по любому полю:
- `?sort_by=name` - сортировка по названию
- `?sort_by=price&order=desc` - сортировка по цене (убывание)
- `?sort_by=volume&order=asc` - сортировка по объему (возрастание)

### Статистика
- **GET /statistics/** - Статистика по всем числовым полям (min, max, avg)
- **GET /statistics/volume** - Статистика по объему
- **GET /statistics/price** - Статистика по цене
- **GET /statistics/stock** - Статистика по количеству

## Установка и запуск

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск сервера
```bash
python main.py
```

Сервер запустится на `http://127.0.0.1:5000`

## Доступ к документации

После запуска сервера доступны:
- **Главная страница**: http://127.0.0.1:5000/
- **Swagger UI**: http://127.0.0.1:5000/apidocs

## Примеры использования

### Получить все напитки
```bash
GET http://127.0.0.1:5000/beverages/
```

### Получить напитки, отсортированные по цене
```bash
GET http://127.0.0.1:5000/beverages/?sort_by=price&order=desc
```

### Добавить новый напиток
```bash
POST http://127.0.0.1:5000/beverages/
Content-Type: application/json

{
  "id": "5",
  "name": "Лимонад",
  "manufacturer": "Фанта",
  "type": "Газированный",
  "volume": 330.0,
  "price": 75.0,
  "stock": 100
}
```

### Получить статистику по цене
```bash
GET http://127.0.0.1:5000/statistics/price
```
